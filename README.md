# Student Grade Management System

A simple console-based Python project to manage student records and grades.

**Developed by:** Rajat  
**Institution:** VIT Bhopal  
**Course:** Computer Science & Engineering

## Features
- Add new student
- View all students
- Update grades
- Calculate average grade
- Find top performer
- Delete student
- Save data permanently

## How to Run
```bash
python main.py
